이미지 첨부 방범

  • src : 이미지 경로 지정
  • alt : 이미지에 대한 설명, 이미지를 표시할 수 없는 경우 보여지는 값

비디오를 삽입했는데 잘 되려나 궁금핑
사진이랑 똑같이 하는데 뒤에 `autoplay controls`를 붙여서 자동 재생 시키는듯?
사진이나 동영상 첨부 방식은 http에서 주소를 복사하거나 내 디렉토리에서 경로 설정하는 방법 두 가지를 쓸 수 있을 것 같다
특성
`autoplay` : 비디오 자동재생, 비활성화 시키려면 ="false"추가해도 안되고 완전히 제거해야함
`controls` : 소리 조정, 동영상 탐색, 일시 정지 등의 컨트롤러 제공
