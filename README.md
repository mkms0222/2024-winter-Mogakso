### 01-19
HTML/CSS 기초 끝내기
#### 태그
**기초**
- `<p>` : 문단
- `<b>` : 굵게
- `<br/>` : 줄바꿈
- `<li>` : 리스트
    - `<oi>` : 정렬된 리스트
    - `<ul>` : 정렬되지 않은 리스트
- `<a>` : 링크
**이미지, 동영상, 오디오**
- 이미지: `<img src="경로 or 주소" alt="설명">`
- 동영상: `<video src="경로 or 주소" 속성>` 
- 오디오: `<audio src="경로 or 주소">`
